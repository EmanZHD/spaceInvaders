### 🎮 **How to Play**  
- **Defeat all aliens** before they reach the bottom.  
- **Score points** by shooting invaders.  
- **Lives**: Lose all lives, and the game ends.  

### 🕹️ **Controls**  
| Action    | Key(s)                     |  
|-----------|----------------------------|  
| Move      | <kbd>←</kbd> <kbd>→</kbd>  |  
| Shoot     | <kbd>Space</kbd>           |  
| Pause     | <kbd>P</kbd> or <kbd>Esc</kbd> |
